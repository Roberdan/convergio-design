# Component semantics

Use this guide when choosing components or when instructing an agent to assemble a page.

Machine-readable exports:
- `@convergio/design-elements/knowledge`
- `@convergio/design-elements/knowledge/components`
- `@convergio/design-elements/knowledge/templates`

## mn-gantt
Use for roadmap, milestone, and dependency-heavy delivery views.

## mn-data-table
Use for dense operational records and sortable business datasets.

## dashboardStrip
Use for an executive KPI strip that compresses gauges, boards, pipelines, and trends.

## mn-detail-panel
Use when users need persistent context without losing their main working surface.
